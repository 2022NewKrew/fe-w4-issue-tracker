### `descripton`

    깃헙의 이슈 관리 시스템을 나만의 스타일로 만들어 현실 이슈를 관리하는 용도로 사용

```
Frontend

├── Emotion
├── Recoil & ReactQuery
└── Webpack & Babel
```

```
Backend
├── MongoDB
├── Express
```
