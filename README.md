# fe-w4-issue-tracker

## Directory Structure
```
📁 backend: Backend source codes and DB are here.
│
├───📁 db: DB source codes and DB files are here.
│
└───📄 app.js: Backend root file.

📁 src: Frontend source codes and files are here.
│
├───📁 component: Components are here. They are hardly reusable.
│
├───📁 item: Reusuable components are here.
│
├───📁 style: css related files are here.
│
├───📁 svg: SVG files are here.
│
├───📄 App.jsx: React root App file.
│
└───📄 index.js: Frontend root file.
```
